## 一、DP核心思想 [⬆️ 返回目录](#toc)

### 1.1 三要素

| 要素 | 含义 | 判断方法 |
|------|------|---------|
| **最优子结构** | 大问题的最优解包含小问题的最优解 | 反证法：假设子问题不是最优，推出矛盾 |
| **无后效性** | 未来状态只与当前状态有关，与到达路径无关 | 检查状态定义是否包含足够信息 |
| **重叠子问题** | 不同大问题共享相同的小问题 | 画递归树看是否有重复计算 |

### 1.2 解题步骤

```
Step 1: 定义状态
        dp[i] 或 dp[i][j] 表示什么含义？

Step 2: 找状态转移方程
        dp[i] = f(dp[i-1], dp[i-2], ...)

Step 3: 确定边界条件
        dp[0] = ?, dp[1] = ?

Step 4: 确定计算顺序
        从小到大？从左上到右下？

Step 5: 输出答案
        dp[n]？max(dp[])？
```

### 1.3 DP vs 递归 vs 贪心

```
┌─────────────────┬─────────────────┬─────────────────┐
│      递归       │      DP         │     贪心      │
├─────────────────┼─────────────────┼─────────────────┤
│ 自顶向下        │ 自底向上        │ 局部最优      │
│ 重复计算        │ 记忆化/填表     │ 不回溯        │
│ 时间爆炸        │ 时间可控        │ 不一定全局最优 │
│ 思路直观        │ 需要找递推      │ 证明困难      │
│ 适合树形        │ 适合序列/网格   │ 适合区间选择  │
└─────────────────┴─────────────────┴─────────────────┘
```

## 二、线性DP [⬆️ 返回目录](#toc)

### 2.1 最长递增子序列 (LIS)

```
状态：dp[i] = 以a[i]结尾的最长递增子序列长度
转移：dp[i] = max(dp[j] + 1)  其中 j < i 且 a[j] < a[i]
边界：dp[i] = 1
时间：O(n²) 朴素 / O(nlogn) 二分优化
```

### 2.2 最长公共子序列 (LCS)

```
状态：dp[i][j] = a[0..i]和b[0..j]的LCS长度
转移：
    if a[i] == b[j]: dp[i][j] = dp[i-1][j-1] + 1
    else: dp[i][j] = max(dp[i-1][j], dp[i][j-1])
边界：dp[0][*] = dp[*][0] = 0
时间：O(n×m)
```

### 2.3 背包问题

| 类型 | 状态定义 | 转移方程 | 时间 |
|------|---------|---------|------|
| **01背包** | dp[i][w] | `max(dp[i-1][w], dp[i-1][w-weight[i]]+value[i])` | O(n×W) |
| **完全背包** | dp[i][w] | `max(dp[i-1][w], dp[i][w-weight[i]]+value[i])` | O(n×W) |
| **多重背包** | dp[i][w] | 二进制拆分转01背包 | O(n×W×logk) |

## 三、区间DP [⬆️ 返回目录](#toc)

### 3.1 核心特征

```
状态含区间：[i, j]
计算顺序：按区间长度从小到大
```

### 3.2 石子合并

```
状态：dp[i][j] = 合并区间[i,j]的最小代价
转移：dp[i][j] = min(dp[i][k] + dp[k+1][j] + sum(i,j))  (i≤k<j)
边界：dp[i][i] = 0
时间：O(n³)
```

## 四、树形DP [⬆️ 返回目录](#toc)

### 4.1 核心思想

```
后序遍历：先算子树，再算根节点
状态：dp[u][0/1] 表示节点u选/不选时的最优值
```

### 4.2 树的最大独立集

```
状态：dp[u][0] = u不选，子树最大独立集大小
       dp[u][1] = u选，子树最大独立集大小
转移：
    dp[u][0] = Σ max(dp[v][0], dp[v][1])  (v是u的子节点)
    dp[u][1] = 1 + Σ dp[v][0]
```

## 五、状态压缩DP [⬆️ 返回目录](#toc)

### 5.1 适用场景

```
状态数量少但维度高：n ≤ 20 左右
用二进制位表示状态：状态 = 0..(2^n-1)
```

### 5.2 旅行商问题 (TSP)

```
状态：dp[S][i] = 已访问集合S，当前在i的最短距离
转移：dp[S][i] = min(dp[S-{i}][j] + dist[j][i])  (j∈S, j≠i)
时间：O(n² × 2^n)
```

## 七、矩阵快速幂优化DP [⬆️ 返回目录](#toc)

### 7.1 适用条件

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   线性递推：f(n) = a₁f(n-1) + a₂f(n-2) + ... + aₖf(n-k)   │
│                                                             │
│   可以用 k×k 变换矩阵 + 矩阵快速幂优化到 O(k³ log n)        │
│                                                             │
│   经典例子：                                                │
│   · 斐波那契：f(n) = f(n-1) + f(n-2)                      │
│   · 爬楼梯：f(n) = f(n-1) + f(n-2)                        │
│   ·  Tribonacci：f(n) = f(n-1) + f(n-2) + f(n-3)         │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### 7.2 变换矩阵构造

```
对于 f(n) = a·f(n-1) + b·f(n-2):

    M = ┌      ┐
        │ a  b │
        │ 1  0 │
        └      ┘

┌          ┐           ┌          ┐
│ f(n)   │             │ f(1)   │
│        │  = M^(n-1) × │        │
│ f(n-1) │             │ f(0)   │
└          ┘           └          ┘
```
