# MT2037 奇怪的和 —— 完整知识点总结

## 一、题目核心

给定 `n×m` 矩阵，求所有**相同值元素对**的曼哈顿距离之和。

曼哈顿距离：`|x₁-x₂| + |y₁-y₂|`

---

## 二、暴力为什么不行

- 最坏情况：所有 `N = n×m` 个元素相同
- 两两组合数：`N(N-1)/2 ≈ 5×10⁹`（当 N=1e5 时）
- 直接遍历超时

**关键洞察**：曼哈顿距离可拆分，x 和 y 方向独立计算。

---

## 三、核心算法：哈希分类 + 前缀和

### 3.1 算法流程

```
1. 展平矩阵：每个元素记录 (值, 行号x, 列号y)
2. 按值排序：相同值聚集在一起
3. 分组处理：对每组提取所有 x 坐标，排序后算距离和
              对每组提取所有 y 坐标，排序后算距离和
4. 累加答案
```

### 3.2 一维距离前缀和公式

对排序后的坐标 `[a₀, a₁, ..., a_{k-1}]`：

第 i 个点与前面 i 个点的距离和 = `a[i] × i - (a[0] + a[1] + ... + a[i-1])`

遍历一遍，维护前缀和即可。

**举例** x 坐标 `[0, 1, 3]`：
- i=0：`0×0 - 0 = 0`，前缀和=0
- i=1：`1×1 - 0 = 1`，前缀和=1（|1-0|=1）
- i=2：`3×2 - 1 = 5`，前缀和=4（|3-0|+|3-1|=3+2=5）
- 总和 = 6

## 四、qsort 完整用法（我之前不会的）

### 4.1 函数原型

```c
#include <stdlib.h>

void qsort(void *base, size_t nmemb, size_t size,
           int (*compar)(const void *, const void *));
```

| 参数 | 含义 |
|------|------|
| `base` | 数组首地址 |
| `nmemb` | 元素个数 |
| `size` | 每个元素的大小（字节） |
| `compar` | 比较函数指针 |

### 4.2 比较函数规则

返回 `负数` → a 排在 b 前面（a < b）  
返回 `0` → a == b  
返回 `正数` → a 排在 b 后面（a > b）

### 4.3 各种类型排序示例

#### 整数数组排序

```c
// 升序
int cmp_int(const void *a, const void *b) {
    return (*(int*)a > *(int*)b) - (*(int*)a < *(int*)b);
}

// 或者简单写法（数据范围小，不会溢出时）
int cmp_int_simple(const void *a, const void *b) {
    return *(int*)a - *(int*)b;
}

int arr[5] = {3, 1, 4, 1, 5};
qsort(arr, 5, sizeof(int), cmp_int);
```

#### 结构体排序（按某个字段）

```c
typedef struct {
    int val, x, y;
} Node;

// 按 val 升序
int cmp_val(const void *a, const void *b) {
    return ((Node*)a)->val - ((Node*)b)->val;
}

// 按 x 升序，x 相同按 y 升序
int cmp_xy(const void *a, const void *b) {
    Node *pa = (Node*)a, *pb = (Node*)b;
    if (pa->x != pb->x) return pa->x - pb->x;
    return pa->y - pb->y;
}

Node nodes[100];
qsort(nodes, n, sizeof(Node), cmp_val);
```

#### 字符串数组排序

```c
int cmp_str(const void *a, const void *b) {
    return strcmp(*(char**)a, *(char**)b);
}

char *words[] = {"banana", "apple", "cherry"};
qsort(words, 3, sizeof(char*), cmp_str);
```

### 4.4 常见坑

| 坑 | 错误 | 正确 |
|----|------|------|
| 比较函数参数 | `int cmp(int a, int b)` | `int cmp(const void *a, const void *b)` |
| 忘记解引用 | `a - b` | `*(int*)a - *(int*)b` |
| 结构体访问 | `a->val`（a是void*） | `((Node*)a)->val` |
| qsort第三个参数 | `sizeof(arr)` | `sizeof(int)` / `sizeof(Node)` |
| 溢出 | `*(int*)a - *(int*)b` | `(*(int*)a > *(int*)b) - (*(int*)a < *(int*)b)` |

## 五、类型转换与溢出防护

### 5.1 为什么要加 `(long long)`

```c
int xs[100005];   // int
int t;            // int

xs[t] * t         // int * int = int，可能溢出！
```

`100000 * 50000 = 5,000,000,000 > INT_MAX(2,147,483,647)`

**溢出发生在赋值给 long long 之前！**

### 5.2 正确写法

```c
ans += (long long) xs[t] * t - pre;   // 先转 long long，再乘法
// 或者
ans += 1LL * xs[t] * t - pre;          // 1LL 强制整个表达式为 long long
```

**注意**：`(long long)(xs[t] * t)` 是错的！先算 int*int（溢出），再转，晚了。

## 六、分组遍历技巧

### 6.1 按值分组（双指针）

```c
int i = 0;
while (i < N) {
    int j = i;
    while (j < N && arr[j].val == arr[i].val) j++;
    // [i, j) 是同一组
    // 处理这组...
    i = j;  // 跳到下一组
}
```

这是 O(N) 的分组方法，比哈希表更简洁（前提是已经按值排序）。

## 七、我的错误汇总

| 错误 | 原因 | 修正 |
|------|------|------|
| `qsort(node, N, sizeof(node), ...)` | `sizeof(node)` 是整个数组 | `sizeof(Node)` |
| `scanf("%d %d", &m, &n)` | 题目先 n 后 m | `scanf("%d %d", &n, &m)` |
| `printf("lld
", ans)` | 少了 `%` | `printf("%lld
", ans)` |
| `return *(int*)a - *(int*)b` | 可能溢出 | `return (*(int*)a > *(int*)b) - (*(int*)a < *(int*)b)` |

## 八、完整代码（修正后）

```c
#include <stdio.h>
#include <stdlib.h>

typedef struct {
    int val, x, y;
} Node;

Node node[100005];
int xs[100005], ys[100005];

int cmp_val(const void* a, const void* b) {
    return ((Node*)a)->val - ((Node*)b)->val;
}

int cmp_int(const void* a, const void* b) {
    return (*(int*)a > *(int*)b) - (*(int*)a < *(int*)b);
}

int main() {
    int n, m;
    scanf("%d %d", &n, &m);

    int N = 0;
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < m; j++) {
            scanf("%d", &node[N].val);
            node[N].x = i;
            node[N].y = j;
            N++;
        }
    }

    qsort(node, N, sizeof(Node), cmp_val);

    long long ans = 0;
    int i = 0;

    while (i < N) {
        int j = i;
        while (j < N && node[j].val == node[i].val) j++;

        int k = j - i;

        // x 方向
        for (int t = 0; t < k; t++) xs[t] = node[i + t].x;
        qsort(xs, k, sizeof(int), cmp_int);
        long long pre = 0;
        for (int t = 0; t < k; t++) {
            ans += (long long) xs[t] * t - pre;
            pre += xs[t];
        }

        // y 方向
        for (int t = 0; t < k; t++) ys[t] = node[i + t].y;
        qsort(ys, k, sizeof(int), cmp_int);
        pre = 0;
        for (int t = 0; t < k; t++) {
            ans += (long long) ys[t] * t - pre;
            pre += ys[t];
        }

        i = j;
    }

    printf("%lld
", ans);
    return 0;
}
```

## 九、复杂度分析

| 指标 | 值 | 说明 |
|------|-----|------|
| 时间 | O(N log N) | 排序 O(N log N) + 分组遍历 O(N) |
| 空间 | O(N) | 存储 N 个节点 + 坐标数组 |

N = n×m ≤ 1e5，完全够用。

---

## 十、一句话总结

> **曼哈顿距离拆 x+y，相同值归类排序，前缀和算距离。qsort 记住 `sizeof(元素)` 不是 `sizeof(数组)`，比较函数参数是 `const void*`。long long 要在乘法前转换，溢出后再转就晚了。**
