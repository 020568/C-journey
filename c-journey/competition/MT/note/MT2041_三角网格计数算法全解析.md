# 📐 璃月海灯节·三角网格计数算法全解析

在 $n$ 等分的大三角形网格中，求其中包含的所有正（▲）和倒（▼）三角形的总数。这是一个经典的从 几何空间拆解 逐步优化到 组合数学/纯递归 的算法题。

## 💡 核心模型认知

**顶点层数 vs 三角形排数**：
- 一个 $d$ 等分的三角形，拥有 $d+1$ 层 横向线条（或顶点层）。
- 但从空间上看，它只包含 $d$ 排 小三角形。

**为什么图论（DFS/找环）不可行**：
- 图论建图不仅难以判断"平行于三条边"的几何约束，而且时间复杂度极高，极易引发 TLE（超时）。
- 正确做法是利用 几何位置关系进行数学递推或递归。

## 🛠️ 解法一：万能组合数学公式（最优解 $O(1)$）

通过对图形中包含的边长为 $k$ 的正倒三角形进行等差数列求和，可以推导出一个无视奇偶性的万能通项公式：

$$T(n) = \lfloor \frac{n(n+2)(2n+1)}{8} \rfloor$$

💡 注：在 C/C++ 中，正整数除法 / 8 本身就自带向下取整（Floor）效果。

### 💻 C 语言实现：

```c
#include <stdio.h>

int main() {
    int N;
    // N 代表测试图案的组数
    if (scanf("%d", &N) != 1) return 0;
    
    // d 代表当前图案的等分数（层数）
    long long d;
    while (N--) {
        scanf("%lld", &d);
        // 使用 long long 防止大数乘法时 int 溢出
        long long ans = d * (d + 2) * (2 * d + 1) / 8;
        printf("%lld\n", ans); // 必须加 \n 换行
    }
    return 0;
}
```

## 🧱 解法二：纯结构递归法（启发式解法 $O(n)$）

如果不记得复杂的数学公式，可以从 大问题化小 的结构视角出发：

一个 $n$ 等分三角形的数量，等于 $n-1$ 等分三角形的数量 加上 最底下新长出来的第 $n$ 排所带来的新三角形。

**递推状态转移方程**：
$$T(n) = T(n-1) + \text{NewUp}(n) + \text{NewDown}(n)$$

- **新正三角 $\text{NewUp}(n)$**：底边必须踩在最底部的全长为 $n$ 的大底边上。数量为等差数列：$1 + 2 + \dots + n = \frac{n(n+1)}{2}$。
- **新倒三角 $\text{NewDown}(n)$**：最下方的顶点（尖尖）必须戳在最底部的底边上。具有完美的间隔递归性：$\text{NewDown}(n) = (n-1) + \text{NewDown}(n-2)$。

### 💻 C 语言实现：

```c
#include <stdio.h>

// 递归计算：由于第 n 排的加入，新增的触底【倒三角】数量
long long get_new_down(long long n) {
    if (n <= 1) return 0; // 边界：0和1等分无法形成倒三角
    return (n - 1) + get_new_down(n - 2);
}

// 核心递归：解出 n 等分图案的总三角形数
long long solve(long long n) {
    if (n == 1) return 1; // 边界：1等分只有1个大三角

    long long old_triangles = solve(n - 1);  // 继承上一层的成果
    long long new_up = n * (n + 1) / 2;      // 这一层新增的正三角
    long long new_down = get_new_down(n);    // 这一层新增的倒三角

    return old_triangles + new_up + new_down;
}

int main() {
    int N;
    if (scanf("%d", &N) != 1) return 0;
    
    long long d;
    while (N--) {
        scanf("%lld", &d);
        printf("%lld\n", solve(d));
    }
    return 0;
}
```

## ⚡ C 语言现实避坑指南（竞赛必看）

| 常见错误 | 错误后果 | 正确姿诀 |
|---------|---------|---------|
| scanf("%d", N) | 段错误 (Runtime Error) | 必须加地址符：scanf("%d", &N) |
| int 连乘大数 | 数据溢出 变成负数 | 变量声明为 long long，读取用 %lld |
| printf("%lld", num) | 答案错误 (WA)（数据粘连） | 结尾必须加换行符：printf("%lld\n", num) |
| 误用 total += num | 答案错误 (WA)（多组数据混淆） | 每组数据独立计算，不要把上一轮的结果滚雪球累加 |

## 🔍 附：while(N--) 的自毁机制

**原理**：后置自减，"先拿当前值做判定，判定完立刻悄悄减 1"。

**效果**：当 N 降到 0 时判定为假，自动跳出循环。是控制测试组数最简洁的"省代码流"写法。
